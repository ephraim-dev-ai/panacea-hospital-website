# 🏥 Panacea Hospital Management System

A modern hospital management system designed to streamline patient care, appointments, and administrative operations.

---

## 🚀 Features

- 👨‍⚕️ Admin Dashboard
- 📅 Appointment Booking System
- 🧾 Patient Records Management
- 📩 Contact & Messaging System
- 🏥 Department & Doctor Management

---

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML, CSS, JavaScript

---

## 📂 Project Structure

/admin → Admin dashboard  
/config → Database connection  
/includes → Layout & helper functions  

---

## ⚙️ Installation

1. Clone the repository  
2. Move to XAMPP `htdocs`  
3. Import `database.sql` into phpMyAdmin  
4. Configure `/config/database.php`  
5. Run project in browser
